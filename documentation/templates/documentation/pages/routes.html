Documentação - Rotas

Accounts

Criar Conta

Descrição: Exibe um formulário para criação de conta no sistema.

Name: accounts:register

URL: 'accounts/register/'

Método: GET

View Correspondente: register_view - Function

Template: accounts/pages/register.html

Variáveis de Contexto:

  • form: Formulário vinculado ao modelo UserCustom
    • Descrição: O formulário que os usuários preencherão para criar uma conta.
    • Tipo: Formulário

POST Criar Conta

Name: accounts:register_create

Descrição: Realiza as validações do formulário de registro e salva o usuário no banco de dados.

URL: 'accounts/register/create/'

Método: POST

View Correspondente: register_create - Function

< h4>Perfil

Descrição: Exibe o perfil do usuário.

Name: accounts:profile

URL: 'accounts/profile'

Método: GET

View Correspondente: ProfileDetail - Class

Template: accounts/pages/profile.html

Variáveis de Contexto:

  • profile: Objeto do modelo CustomUser
    • Descrição: O objeto do usuário personalizado associado ao perfil.
    • Tipo: CustomUser
  • profile_page: Booleano indicando que estamos na página de profile.
    • Descrição: Pode ser usado para realçar visualmente a guia de perfil no menu de navegação.
    • Tipo: bool
  • profile_tab: Booleano indicando que estamos na aba de profile.
    • Descrição: Pode ser usado para destacar a aba de perfil no layout da página.
    • Tipo: bool
  • categories: Lista de todas as categorias cadastradas no sistema.
    • Descrição: A lista de categorias disponíveis para exibição das categorias selecionadas ou não selecionadas
    • Tipo: Lista de Category

Meus Projetos

Descrição: Exibe os projetos em que o usuário está inserido, seja como dono, moderador ou membro.

Name: accounts:my_projects

URL: 'accounts/profile/projects'

Método: GET

View Correspondente: MyProjectsList - Class

Template: accounts/pages/my_projects.html

Variáveis de Contexto:

  • projects: Projetos
    • Descrição: Os projetos em que o usuário está inserido como dono, moderador ou membro.
    • Tipo: Lista de Project
  • profile_page: Booleano indicando que estamos na página de perfil.
    • Descrição: Pode ser usado para realçar visualmente a guia de perfil no menu de navegação.
    • Tipo: bool
  • project_tab: Booleano indicando que estamos na aba de projetos.
    • Descrição: Pode ser usado para destacar a aba de projetos no layout da página.
    • Tipo: bool
  • profile: Objeto do modelo CustomUser
    • Descrição: O objeto do usuário personalizado associado ao perfil.
    • Tipo: CustomUser
